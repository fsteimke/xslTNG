# DocBook für die KoSIT
Dies  wird ein  Oxygen Framework  für die  Arbeit mit  DocBook in  der
KoSIT.   Es basiert  auf den  von N.   Walsh neu  entwickelten *xslTNG
Stylsheets*, die  erheblich komfortabler  sind und  mehr Möglichkeiten
bieten als die bisher genutzten.

# xslTNG und xslTNG fork

The *The Next Generation* of DocBook stylesheets in XSLT is a complete
reimplementation     of    the     stylesheets    for     transforming
[DocBook](https://docbook.org/) into modern,  clean, semantically rich
HTML. The presentation  is supported by CSS and (if  you wish, a small
amount of) JavaScript.  The expectation for paginated output is to use
HTML+CSS.

Die Entwicklung der Original Stylesheets erfolgt durch Norman Walsh in
einem GitHub Projekt
[https://xsltng.docbook.org/](https://xsltng.docbook.org/).
Die zugehörige Dokumentation ist der [DocBook xslTNG
Reference Guide](https://xsltng.docbook.org/guide/). 

Zur Beteiligung  an der Entwicklung  (in Form von Pull  Requests) habe
ich        [ein        eigenes        Projekt        als        *Fork*
eingerichtet](https://github.com/fsteimke/xslTNG). Dort  wird in einem
eigenen Branch *kosit* auch  das Oxygen Framework vorbereitet, welches
die notwendigen Anapssungen für die KoSIT enthält.
